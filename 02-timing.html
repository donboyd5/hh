7  When do people come to Hubbard Hall?

When the same production plays Friday, Saturday, and Sunday, which day do people pick? This chapter leads with that within-run view — the cleanest read on audience day preference, since it holds the show constant — then asks how the answer has changed over time, whether opera and music runs behave the same way, and closes with the coarser all-events view. How runs are reconstructed from Neon events, what is excluded, and how headcount is counted are in the Methodology appendix.

7.1 Within a theater run: Friday, Saturday, or Sunday?

Only theater production runs that played all three of Friday, Saturday, and Sunday — mostly mainstage shows running one to four weekends. Attendance is true headcount, not registrations.

7.2 Has the best day changed over time?

The numbers above pool 2013–2026, but the day ranking has not been stable. Still holding the product constant (full-weekend theater runs, attendees per performance so days with two shows don’t get double credit), here is the same question across roughly five-year periods.

Code 
PERIODS = ["2013–2015", "2016–2020", "2021–2026"]

def period(year):
    return pd.cut(year, bins=[2010, 2015, 2020, 2026], labels=PERIODS)

run_rate = (
    wk.groupby(["production_run", "weekday"], as_index=False)
    .agg(performances=("attendees", "count"), attendees=("attendees", "sum"))
)
run_rate = run_rate[run_rate["production_run"].isin(charted)].copy()
run_rate["rate"] = run_rate["attendees"] / run_rate["performances"]
run_rate["run_start"] = run_rate["production_run"].map(
    wk.groupby("production_run")["starts_on"].min()
)

recent = run_rate[run_rate["run_start"] >= "2021-01-01"]
best_day = recent.loc[recent.groupby("production_run")["rate"].idxmax(), "weekday"]
n_recent = recent["production_run"].nunique()
n_recent_sun = int((best_day == "Sunday").sum())

wk_periods = wk.assign(period=period(wk["starts_on"].dt.year))
pd_rate = (
    wk_periods.groupby(["period", "weekday"], observed=True)
    .agg(performances=("attendees", "count"), attendees=("attendees", "sum"))
    .reset_index()
)
pd_rate["rate"] = pd_rate["attendees"] / pd_rate["performances"]
rate_of = lambda p, d: pd_rate.set_index(["period", "weekday"])["rate"].loc[(p, d)]

Per-performance attendance has roughly doubled across the periods, and the day ranking has flipped. In 2013–2015 Friday was the strongest day (27 per show vs 24 on Sunday); in 2016–2020 the days were about even; since 2021 Sunday is clearly strongest — 66 per show vs about 50 on Friday and Saturday. Sunday has been the best-attended day in 9 of the 9 full-weekend runs since 2021.

Recent-period caveats: 2021–2026 rests on 9 runs (52 performances), and recent runs rarely schedule Saturday matinees, which used to pull Saturday’s average down — yet Sunday leads anyway.

7.3 Do opera and music runs behave the same way?

Music and opera get the same treatment: map event names to productions, keep runs that played all of Friday, Saturday, and Sunday. Almost all music events are one-off concerts (Music from Salem plays single afternoons, cabarets play one or two nights), so nearly everything that qualifies is opera — the summer festival stagings of 2013–2017, which also played real midweek (Wednesday/Thursday) performances — plus one 2015 musical folktale with a theater-style four-weekend run. No over-time comparison is possible: the qualifying runs all fall in 2013–2017.

7.4 Which days are busiest overall?

The coarser view: every event of every kind, whether or not it was part of a run. Counts here are registrations (a proxy for headcount), and because most events are scheduled on the day the organizers chose, this mostly reflects what gets booked rather than audience preference — use the within-run sections above for that. It is kept for context.

Code 
cat_order = ["theater", "music", "dance", "opera", "class"]
named = regs[regs["event_minorcat"].isin(cat_order)].assign(
    cat=lambda d: d["event_minorcat"]

combo = pd.concat([named, regs[regs["event_majorcat"] == "class"].assign(cat="class")])

Theater peaks Saturday; music peaks Sunday. Classes spread across the week with a Friday peak.
