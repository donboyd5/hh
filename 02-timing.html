3  When people come

Which days of the week draw the most attendance? Counts below are registrations (a proxy for headcount), grouped by the event’s performance date. Each panel is one kind of programming.

Code 
import pandas as pd
import plotly.express as px
from hh.io import read_parquet
from hh.viz import style, BLUE
from hh.analytics.timing import WEEKDAY_ORDER

regs = read_parquet("processed", "registrations_enriched.parquet")
cat_order = ["theater", "music", "dance", "opera", "class"]

named = regs[regs["event_minorcat"].isin(cat_order)].assign(
    cat=lambda d: d["event_minorcat"]
)
combo = pd.concat([named, regs[regs["event_majorcat"] == "class"].assign(cat="class")])
combo = combo.dropna(subset=["weekday"])
tab = combo.groupby(["cat", "weekday"], observed=True).size().reset_index(name="attendance")

3.1 Within a run: Friday, Saturday, or Sunday?

The panels above pool every theater event, including one-night shows, so a “Saturday peak” could just mean more shows are booked on Saturdays. A sharper version of the question: when the same production plays Friday, Saturday, and Sunday, which day do people pick?

Neon stores each performance as a separate event (e.g. “Fun Home - Friday, February 3 at 7:30 pm”) with no field linking the nights of a run, so event names were manually mapped to productions (hh.analytics.productions), and a production’s performances are split into separate runs when more than 45 days apart. This section keeps only production runs that played all three of Friday, Saturday, and Sunday — mostly mainstage shows running one to four weekends.

Two other differences from the counts above: attendance here is true headcount (ticket attendees whose registration succeeded, not registrations — one registration averages about two people), and canceled or moved performances, open rehearsals, online productions, and weekday 10 am school shows are excluded.

Code 
from hh.viz import CATEGORICAL
from hh.analytics.productions import production_performances, full_weekend_runs, time_slot

perf = production_performances(regs, minorcats=("theater",))
fw = full_weekend_runs(perf)

day_order = [d for d in WEEKDAY_ORDER if d in set(fw["weekday"])]
day = (
    fw.groupby("weekday")
    .agg(performances=("swept_event_id", "count"), attendees=("attendees", "sum"))
    .reindex(day_order)
)
day["attendees per performance"] = day["attendees"] / day["performances"]
day["share of attendees"] = day["attendees"] / day["attendees"].sum()

n_runs = fw["production_run"].nunique()
yr_span = f"{fw['starts_on'].min():%Y}–{fw['starts_on'].max():%Y}"
sat_share = f"{day.loc['Saturday', 'share of attendees']:.0%}"
per_perf = {
    d: f"{day.loc[d, 'attendees per performance']:.0f}"
    for d in ("Friday", "Saturday", "Sunday")
}

Saturday evenings draw 39 attendees per show versus 29 for Saturday matinees — the weakest slot of the weekend. Sunday’s single 2 pm show (37 per show) draws as well as any slot, so the matinee habit is a Sunday habit, not a Saturday one.

3.1.1 Do opera and music runs behave the same way?

Music and opera get the same treatment: map event names to productions, keep runs that played all of Friday, Saturday, and Sunday. Almost all music events are one-off concerts (Music from Salem plays single afternoons, cabarets play one or two nights), so nearly everything that qualifies is opera — the summer festival stagings of 2013–2017, which also played real midweek (Wednesday/Thursday) performances — plus one 2015 musical folktale with a theater-style four-weekend run.

Code 
mo_fw = full_weekend_runs(production_performances(regs, minorcats=("music", "opera")))

mo_day_order = [d for d in WEEKDAY_ORDER if d in set(mo_fw["weekday"])]

    mo_fw.groupby("weekday")

Opera audiences didn’t privilege the weekend: the midweek opera nights drew the most people per performance (Thursday 76, versus 49 on Friday and 38 on Saturday, the weakest day per show — diluted, as in theater, by Saturday matinees). The sample is small (a few midweek performances, all in the 2013–2017 festival era), so treat the midweek numbers as suggestive rather than precise.

The per-performance numbers above pool 2013–2026, but the day ranking has not been stable — Sunday has become the strongest day in recent years. That story is in How things changed over time. (That trend analysis is theater-only: the qualifying opera and music runs all fall in 2013–2017, too narrow a window for an over-time comparison.)
