4  Are large donors going away?

“Large” here means a household that gave $1,000 or more in a single year. This chapter tracks how many such households are active each year, how many donors come back year to year, and how many once-active donors have gone quiet. Definitions and the retention method are in Methodology; the size tiers are introduced in How has giving by donation size changed over time?.

Code 
import pandas as pd
import plotly.express as px
from hh.io import read_parquet
from hh.clean.accounts import clean_accounts
from hh.clean.donations import clean_donations
from hh.analytics.donors import (
    annual_gifts,
    donor_retention,
    donors_above_threshold_by_year,
    size_tier,
    TIER_LABELS,
    LARGE_DONOR_MIN,
)
from hh.viz import style, BLUE, pct

ag = annual_gifts(clean_donations(accounts=clean_accounts()))
hh = read_parquet("processed", "households_summary.parquet")

max_year = int(ag["year"].max())                 # current, partial year
complete_below = [y for y in sorted(ag["year"].unique()) if y < max_year]

# (a) households reaching the large-donor threshold each year
big = donors_above_threshold_by_year(ag, min_amount=LARGE_DONOR_MIN)
big_plot = big[big["year"].isin(complete_below)]
big_ytd = int(big.loc[big["year"] == max_year, "donors"].sum())

# (b) year-over-year retention. Drop the row whose "next year" is the partial year.
ret = donor_retention(ag)
ret_plot = ret[ret["year"] < max_year - 1].copy()

# (c) lapsed donors: ever gave, but not in the last two complete years
recent_complete = [max_year - 1, max_year - 2]
ever = hh[hh["is_donor"]].copy()
recent_ids = set(ag.loc[ag["year"].isin(recent_complete), "id"])
ever["lapsed"] = ~ever["id"].isin(recent_ids)
lapsed = ever[ever["lapsed"]]
n_lapsed = len(lapsed)
lapsed_large = int((lapsed["total_donations"] >= LARGE_DONOR_MIN).sum())
lapsed["tier"] = lapsed["total_donations"].map(size_tier)
lapsed_by_tier = lapsed.groupby("tier")["id"].count().reindex(TIER_LABELS).fillna(0).astype(int)

4.3 Who has gone quiet?

Of the 1,298 households that have ever given, 794 have not given in either of the last two complete years (2024 or 2025). Of those lapsed households, 69 were once large donors (gave $1,000+ in a year at some point). By lifetime giving tier:

Code 
ldf = lapsed_by_tier.reset_index()
ldf.columns = ["Lifetime giving tier", "Lapsed households"]
ldf["Share of lapsed"] = ldf["Lapsed households"] / n_lapsed
ldf.style.format({"Lapsed households": "{:,.0f}", "Share of lapsed": pct})
  Lifetime giving tier Lapsed households Share of lapsed
0 <$100 446 56.2%
1 $100–999 279 35.1%
2 $1,000–4,999 56 7.1%
3 $5,000+ 13 1.6%

2026 is year-to-date, so it is left out of the trends. So far in 2026, 4 households have reached the $1,000+ threshold; the full-year count will be higher. Retention is an upper bound on voluntary churn — a donor who has died also appears as “not retained.”
