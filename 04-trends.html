5  How things changed over time

Two simple views across roughly five-year periods (2013–2015, 2016–2020, 2021–2026): where donor dollars come from, and which days draw attendance. Both are shown as shares within each period so changes in the mix are visible regardless of overall growth.

Code 
import pandas as pd
import plotly.express as px
from hh.io import read_parquet
from hh.clean.accounts import clean_accounts
from hh.clean.donations import clean_donations
from hh.analytics.donors import succeeded_individual_gifts
from hh.analytics.timing import WEEKDAY_ORDER
from hh.viz import style, CATEGORICAL

PERIODS = ["2013–2015", "2016–2020", "2021–2026"]
BANDS = ["[0, 1)", "[1, 5)", "[5, 10)", "[10, 20)", "[20, ∞)"]
BAND_COLORS = dict(zip(BANDS, ["#b7d3f6", "#6da7ec", "#3987e5", "#184f95", "#0d366b"]))

def period(year):
    return pd.cut(
        year, bins=[2010, 2015, 2020, 2026], labels=PERIODS
    )

# donor dollars by distance band x period
accounts_geo = read_parquet("processed", "accounts_geocoded.parquet")
gifts = succeeded_individual_gifts(clean_donations(accounts=clean_accounts()))
gifts["distance_band"] = gifts["account_id"].map(
    accounts_geo.set_index("account_id")["distance_band"]
)
gifts["period"] = period(gifts["donation_date"].dt.year)
gd = (
    gifts.dropna(subset=["distance_band", "period"])
    .groupby(["period", "distance_band"], observed=True)["donation_amount"]
    .sum()
    .reset_index()
)
gd["share"] = gd.groupby("period", observed=True)["donation_amount"].transform(
    lambda s: s / s.sum()
)

5.1 Fri vs Sat vs Sun within theater runs, over time

The figure above pools every event, so it mostly reflects what gets scheduled — more Saturday events means more Saturday attendance. To ask whether the audience’s day preference has shifted, this section holds the product constant: only theater production runs that played all three of Friday, Saturday, and Sunday (reconstructed by the manual event-name mapping described in When people come), measured as attendees per performance so days with two shows don’t get double credit. Headcount is ticket attendees whose registration succeeded.

Code 
from hh.analytics.productions import production_performances, full_weekend_runs

fw = full_weekend_runs(
    production_performances(read_parquet("processed", "registrations_enriched.parquet"))
)
wk = fw[fw["weekday"].isin(["Friday", "Saturday", "Sunday"])].copy()
DAY_COLORS = {"Friday": CATEGORICAL[0], "Saturday": CATEGORICAL[1], "Sunday": CATEGORICAL[2]}

run_day = (
    wk.groupby(["production_run", "weekday"], as_index=False)
    .agg(performances=("starts_on", "count"), attendees=("attendees", "sum"))
)
run_day["rate"] = run_day["attendees"] / run_day["performances"]
run_day["run_start"] = run_day["production_run"].map(
    wk.groupby("production_run")["starts_on"].min()
)
run_totals = wk.groupby("production_run")["attendees"].sum()
run_day = run_day[run_day["production_run"].map(run_totals) >= 30]

recent = run_day[run_day["run_start"] >= "2021-01-01"]
best_day = recent.loc[recent.groupby("production_run")["rate"].idxmax(), "weekday"]
n_recent = recent["production_run"].nunique()
n_recent_sun = int((best_day == "Sunday").sum())

wk["period"] = period(wk["starts_on"].dt.year)
pd_rate = (
    wk.groupby(["period", "weekday"], observed=True)
    .agg(performances=("starts_on", "count"), attendees=("attendees", "sum"))
    .reset_index()
)
pd_rate["rate"] = pd_rate["attendees"] / pd_rate["performances"]
rate_of = lambda p, d: pd_rate.set_index(["period", "weekday"])["rate"].loc[(p, d)]

Per-performance attendance has roughly doubled across the periods, and the day ranking has flipped. In 2013–2015 Friday was the strongest day (27 per show vs 24 on Sunday); in 2016–2020 the days were about even; since 2021 Sunday is clearly strongest — 66 per show vs about 50 on Friday and Saturday. Sunday has been the best-attended day in 9 of the 9 full-weekend runs since 2021.

Recent-period caveats: 2021–2026 rests on 9 runs (52 performances), and recent runs rarely schedule Saturday matinees, which used to pull Saturday’s average down — yet Sunday leads anyway.
