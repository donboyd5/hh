5  What do large donors attend?

Donors are split by lifetime giving — smaller donor (under $1,000 lifetime), large donor ($1,000–4,999), and major donor ($5,000+) — and compared with households that have never given. The question is whether larger donors engage differently: do they attend more, take more classes, and prefer different performance genres? Attendance is headcount (succeeded ticket attendees); see Methodology. Size tiers are defined in How has giving by donation size changed over time?; the binary donor view is in What performances do donors attend?.

Code 
import pandas as pd
import plotly.express as px
from hh.io import read_parquet
from hh.viz import style, BLUE, CATEGORICAL, SEQUENTIAL_BLUE, pct
from hh.analytics.productions import is_excluded, count_succeeded_attendees

GROUP_ORDER = [
    "Smaller donor (<$1,000)",
    "Large donor ($1,000–4,999)",
    "Major donor ($5,000+)",
    "Non-donor",
]
GROUP_COLORS = {
    "Smaller donor (<$1,000)": SEQUENTIAL_BLUE[3],
    "Large donor ($1,000–4,999)": SEQUENTIAL_BLUE[8],
    "Major donor ($5,000+)": SEQUENTIAL_BLUE[12],
    "Non-donor": "#c9c7c0",
}
GENRES = ["theater", "music", "opera", "dance"]
GENRE_COLORS = dict(zip(GENRES, CATEGORICAL[:4]))

hh = read_parquet("processed", "households_summary.parquet")
regs = read_parquet("processed", "registrations_enriched.parquet")


def _lifetime_group(r):
    if not r["is_donor"]:
        return "Non-donor"
    t = r["total_donations"]
    if t >= 5000:
        return "Major donor ($5,000+)"
    if t >= 1000:
        return "Large donor ($1,000–4,999)"
    return "Smaller donor (<$1,000)"


hh["lt_group"] = hh.apply(_lifetime_group, axis=1)

5.2 Which performances draw the large-donor tiers?

Performance headcount by genre, split by lifetime-donor tier:

Code 
m = regs.merge(hh[["id", "lt_group"]], on="id", how="left")
perf = m[(m["event_majorcat"] == "performance") & ~m["event_name"].map(is_excluded)].copy()
perf["attendees"] = perf["tickets"].map(count_succeeded_attendees)

gxg = perf.groupby(["lt_group", "event_minorcat"])["attendees"].sum().unstack(fill_value=0)
gxg = gxg.reindex(columns=GENRES).fillna(0)
genre_mix = gxg.div(gxg.sum(axis=1), axis=0).reindex(GROUP_ORDER)

Major donors are the most likely to attend performances (92% of them are patrons) and the most likely to take classes (65%). The genres they attend mirror the rest of the audience — theater first — but opera and music take a noticeably larger share of their attendance than of non-donors’.
